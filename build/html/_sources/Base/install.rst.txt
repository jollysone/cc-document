####################################################################################################
**安装**
####################################################################################################

:获取方式:

.. code-block:: bash

    > git clone http://git.xbwq.com.cn/framework/CC2

:选择分支:

根据相应项目切换到对应的分支即可。

:注意事项:

目前尚不支持 ``composer`` 方式获取。
