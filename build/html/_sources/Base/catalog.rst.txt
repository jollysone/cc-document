####################################################################################################
**目录结构**
####################################################################################################

.. code-block:: 

    project  应用部署目录
    ├─protected      应用目录
    │  ├─biz              
    │  ├─module         应用模块
    │  ├─pre            
    │  └─runtime        应用的运行时目录
    ├─public         WEB 相关内容目录
    │  ├─common         链接必须的框架基础web文件
    │  └─plug           链接必须的框架扩展web文件
    ├─.gitignore     用于 忽略Git文件提交
    ├─.htaccess      用于 apache 的重写
    └─index.php      项目入口文件

    CC flamework  框架部署目录（默认与 project  应用部署目录同级/可自定义修改相关配置文件）
